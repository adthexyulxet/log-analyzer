# EDR 로그 분석기 (Log Analyzer)
EDR 로그(`.tar.gz`)에서 주요 에러 또는 필요한 로그를 검색하여 HTML 보고서 형태로 작성해 주는 프로그램입니다.

# 주요 기능
1. **자동 로그 분석**: SentinelOne에서 자주 발생하는 주요 에러(12종)를 자동으로 탐지합니다.
2. **로그 검색**: 최대 5개까지 사용자가 직접 키워드를 입력하여 로그를 검색할 수 있습니다. (대소문자 구분 없음)
3. **EDR 부하 분석**: 리소스 부하 추이, 최다 발생 프로세스 등 EDR 부하 관련 지표를 확인할 수 있습니다.
4. **HTML 보고서 생성**: 검색된 로그를 html 문서로 저장 및 출력합니다.
5. **로그 다운로드**: 매칭된 전체 로그를 CSV 또는 TXT 파일로 다운로드할 수 있습니다.

# 사용 방법
1. 프로그램을 실행하고 분석할 `.tar.gz` 로그 파일을 선택합니다.
2. 분석 모드를 선택합니다.
3. 분석이 완료되면 Log Analyzer와 동일 경로에 `(분석모드)_(원본파일명).html` 결과 파일이 생성되며 자동으로 열립니다.
